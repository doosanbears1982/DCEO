# 📈 StockBoard: 초보자를 위한 글로벌 주식 현황판

**StockBoard**는 복잡한 주식 시장의 흐름을 한눈에 파악할 수 있도록 설계된 **종합 주식 정보 대시보드**입니다. 주식을 처음 시작하는 입문자부터 숙련된 투자자까지, 여러 사이트를 돌아다닐 필요 없이 이 페이지 하나로 시장의 맥락을 짚을 수 있습니다.

---

## 🌟 주요 기능 안내

### 1. 실시간 시장 지수 (Market Overview)
*   **미국/아시아/암호화폐 탭**: 전 세계 주요 시장의 현재 상태를 30초마다 자동으로 업데이트하여 보여줍니다.
*   **지수 해석**: 숫자가 **빨간색**이면 전날보다 상승, **파란색**이면 하락을 의미합니다. 시장이 전반적으로 어떤 분위기인지 가장 먼저 확인하세요.

### 2. 시장 트렌드 (Market Trends)
*   **거래 상위**: 현재 사람들이 가장 많이 사고파는 인기 종목입니다.
*   **최대 상승/하락**: 오늘 하루 동안 가장 드라마틱하게 가격이 변한 종목들을 순위별로 보여줍니다.

### 3. 인터렉티브 주가 차트 (Interactive Charts)
*   **기간별 보기**: 1일, 1개월, 1년, 전체 기간 버튼을 눌러 주가가 과거부터 어떻게 흘러왔는지 그래프로 확인하세요.
*   **거래량 확인**: 차트 하단에 막대그래프로 표시되는 거래량은 해당 가격대에서 얼마나 많은 거래가 일어났는지 알려줍니다.

### 4. 기업 재무제표 시각화 (Financial Statements)
*   **성장성 확인**: 어려운 숫자 대신 **매출액, 영업이익, 당기순이익**을 막대그래프로 보여줍니다. 막대가 매년 높아지고 있다면 성장하고 있는 좋은 기업일 가능성이 높습니다.
*   **핵심 지표**: PER(주가수익비율), PBR(주가순자산비율) 등 어려운 용어도 한곳에서 확인할 수 있습니다.

### 5. 나만의 관심 종목 (Watchlist)
*   **별(⭐) 표시**: 관심 있는 종목 옆의 별 아이콘을 누르면 왼쪽 사이드바에 저장됩니다. 언제든 다시 들어왔을 때 클릭 한 번으로 상세 정보를 볼 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

이 프로젝트는 최신 웹 기술을 사용하여 빠르고 안정적으로 작동합니다.
*   **Frontend**: Next.js (React), Tailwind CSS (깔끔한 디자인)
*   **Charts**: Recharts (데이터 시각화)
*   **Data API**: Yahoo Finance API (실시간 주가 연동)
*   **Icons**: Lucide React

## 🚀 시작하기 (Setup Guide)

컴퓨터에 Node.js가 설치되어 있다면 아래 순서대로 실행할 수 있습니다.

1.  **저장소 복제 및 이동**
    ```bash
    git clone https://github.com/doosanbears1982/DCEO.git
    cd DCEO/stockboard
    ```

2.  **필요한 라이브러리 설치**
    ```bash
    npm install
    ```

3.  **프로그램 실행**
    ```bash
    npm run dev
    ```

4.  **브라우저 접속**
    웹 브라우저 주소창에 `http://localhost:3000`을 입력하면 대시보드가 나타납니다!

## 💡 투자 초보자를 위한 팁
*   **검색창 활용**: `AAPL`(애플), `NVDA`(엔비디아), `005930.KS`(삼성전자) 처럼 종목 코드(티커)를 입력하면 해당 기업의 모든 정보를 볼 수 있습니다.
*   **다크 모드**: 밤에 차트를 볼 때는 상단의 **달 아이콘**을 눌러보세요. 눈이 훨씬 편해집니다.
*   **뉴스 피드**: 종목을 클릭하면 하단에 해당 기업과 관련된 최신 뉴스만 모아서 보여줍니다. 호재나 악재를 빠르게 파악해 보세요.

---

**면책 조항**: 본 서비스에서 제공하는 모든 데이터는 정보 제공용이며, 실제 투자 시 발생하는 결과에 대한 책임은 투자자 본인에게 있습니다. 실제 거래 전에는 반드시 정식 증권사 앱(MTS/HTS)을 통해 가격을 재확인하시기 바랍니다.
