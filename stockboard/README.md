# StockBoard (주식 현황판)

종합 주식 정보 대시보드 웹 애플리케이션입니다.

## 핵심 기능
- **시장 뷰어**: S&P 500, NASDAQ, KOSPI 등 주요 지수 실시간 모니터링
- **인터렉티브 차트**: 일간, 월간, 연간 주가 추이 및 거래량 시각화
- **기업 분석**: 핵심 재무 지표(매출, 영업이익 등) 막대 그래프 제공
- **맞춤형 뉴스**: 종목별 관련 최신 뉴스 실시간 피드
- **관심 종목**: 로컬 스토리지를 활용한 나만의 포트폴리오 관리

## 기술 스택
- Next.js (App Router)
- Tailwind CSS
- Recharts (Data Visualization)
- Lucide React (Icons)
- Yahoo Finance API (Server-side Integration)

## 시작하기
```bash
cd stockboard
npm install
npm run dev
```
